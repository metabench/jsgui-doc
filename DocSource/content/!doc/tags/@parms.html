<@parms> tag

Example:
<@codeblock>
Parameters:
<@parms> <@parm name="obj" type="object" description="Fields initialization object" optional /> <@parm name="data_def" type="?" description="Not used in fact" optional />
Result:
Description:

Describes method parameters using nested <@parm> tags.

The <@parm> tag attributes are the following:

name
Parameter name
type
Parameter type
description
Parameter description
default
Parameter default value (if present)
optional
Marks this parameter as optional
