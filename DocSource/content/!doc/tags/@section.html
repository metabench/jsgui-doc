<@section> tag

The <@section> tag generates an expandable block (initially expanded) for a long page section (e.g. group of methods).

Example:
<@codeblock> <@section title="Fields"> <@item title="set_field(...)">

set_field(...) description

<@item title="field()">

field() description

Result (live):
<@section title="Fields"> <@item title="set_field(...)">

set_field(...) description

<@item title="field()">

field() description

Attributes:
title

The section title, e.g. "Fields".

id (optional)

Id attribute for the generated HTML header element.
