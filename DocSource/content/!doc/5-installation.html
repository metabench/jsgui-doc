GenDoc.exe install

  1. Install Mocha test framework globally: mochajs.org
  2. Open the global npm directory, node_modules subdirectory. Create "my-mocha-reporter" subdir inside the node_modules dir.
  3. Open the doc/Doc.sln Visual Studio solution from the doc repository.
  4. Open the my-mocha-reporter project properties, select the "Build Events" tab. On the "Post-build event command line" box update the source and destination pathes on the following manner: the copy commands must copy 2 files from the my-mocha-reporter project to the my-mocha-reporter npm directory created at step 2.
  5. Copy all from assets directory from the my-mocha-reporter project to the my-mocha-reporter npm directory created at step 2.
  6. Build the my-mocha-reporter project. Check the npm my-mocha-reporter directory, it should contains the following: <@codeblock> node_modules (7 dirs inside) .npmignore main.js package.json page-test-template.html Readme.md
  7. Update the settings.xml file in the GenDoc project.
  8. Build the GenDoc project.
  9. Run GenDoc.exe, check the output directories.
