Module: core/jsgui-lang-enh

core/jsgui-lang-enh

Language enhance module. Just adds a fromObject() method to the jsgui-lang-essentials module.

<@section title="Members"> <@item title="fromObject(value)">

Creates and returns an enhanced value class instance for the passed value. In other words, it wraps the value into an enhanced class. Depending of the value type, the result will be following:

  • array: new Collection(), and the collection items are the array items passed through fromObject() in turn
  • object: new Enhanced_Data_Object(), and the Enhanced_Data_Object values are object fields passed through fromObject() in turn
  • string: new Data_Value()
  • number: new Data_Value()
  • other: undefined
Parameters:
<@parms> <@parm name="value" type="*" description="value(s) to wrap" />
Example
<@codeblock> jsgui_enh.fromObject(111) ==> new Data_Value({ 'value': 111 }) jsgui_enh.fromObject("22") ==> new Data_Value({ 'value': "22" }) // var edo = jsgui_enh.fromObject({ Field1: 111, Field2: "22" }); // the fromObject() call above is equivalent to the following: // var edo = new Enhanced_Data_Object(); // edo.set("Field1", new Data_Value({ 'value': 111 })); // edo.set("Field2", new Data_Value({ 'value': "22" })); // var collection = jsgui_enh.fromObject([111, "22"]); // the fromObject() call above is equivalent to the following: // var collection = new Collection(); // collection.push(new Data_Value({ 'value': 111 })); // collection.push(new Data_Value({ 'value': "22" })); <@item title="Side effects" class="remark"> <@codeblock> var jsgui = require('./jsgui-lang-util'); // jsgui.Enhanced_Data_Object = Enhanced_Data_Object; jsgui.Mini_Context = Enhanced_Data_Object.Mini_Context; // jsgui.fromObject = fromObject;
