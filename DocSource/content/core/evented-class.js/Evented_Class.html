Class: Evented_Class

core/evented-class.js. Evented_Class

Represents a class supporting events.

new Evented_Class()

Creates the Evented_Class object.

Methods

add_event_listener(fn_listener)

Adds a general event listener (calling for all events).

The raising event name will be passed to the listener (prior to possible other arguments).

Parameters:
Name Type Description
fn_listener Function Event listener (handler) function
Example
buttonOk.add_event_listener(function(event_name){ alert(event_name + " raised!"); });

add_event_listener(event_name, fn_listener)

Adds the fn_listener event listener to the event_name event listeners queue.

Parameters:
Name Type Description
event_name String Event name
fn_listener Function Event listener (handler) function
Example
buttonOk.add_event_listener("click", function(){ alert("click event"); });

on(event_name, fn_listener)

A shortcut for the add_event_listener(listener) or add_event_listener(event_name, listener) method (depending of the passed parameters).

Example
<@codeblock> buttonOk.on(function(){ alert("some event"); }); buttonOk.on("click", function(){ alert("click event"); });

remove_event_listener(event_name, fn_listener)

Removes the fn_listener event listener from the event_name event listeners queue.

Parameters:
Name Type Description
event_name String Event name
fn_listener Function Event listener (handler) function
Example
var click_handler = function(){ console.log("clicked!"); };
buttonOk.add_event_listener("click", click_handler);
buttonOk.raise_event("click");

buttonOk.remove_event_listener("click", click_handler);

off(event_name, fn_listener)

A shortcut for the remove_event_listener() method.

Example
buttonOk.off("click", ok_click_handler);

one(event_name, fn_listener)

Adds the fn_listener event listener to the event_name event listeners queue. The listener will be called one time only (when the event will be raised next time), then the listener will be automatically removed from the queue.

The listener's return value will be lost (raise_event() will add undefined to the returning array.)

The listener will get only first parameter from the additional parameters passed to the raise_event() method.)

Parameters:
Name Type Description
event_name String Event name
fn_listener Function Event listener (handler) function
Example
buttonOk.one("click", function(){ console.log("clicked!"); });

raise_event(event_name, additional_args...)

Raises the event event_name. If additional_args... are passed, then the additional arguments are passed to the event listeners.

For the general event listeners (added using add_event_listener(listener) method) the event_name value passes to the listeners prior to other arguments.

Returning value is an array containing all the called listeners return values.

Parameters:
Name Type Description
event_name String Event name
additional_args * Possible additional arguments
Example
buttonOk.raise_event("click");

raise()

A shortcut for the raise_event() method.

Example
buttonOk.raise("click");

trigger()

A shortcut for the raise_event() method.

Example
buttonOk.trigger("click");
