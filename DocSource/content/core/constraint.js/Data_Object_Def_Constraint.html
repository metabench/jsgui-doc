Parameters:
Name Type Description
value object

object to check
Example
var c = new Constraint.Data_Object_Def_Constraint({ name: "string", age: "number" });
c.match({ name: "John", age: 25 })  ==> true
c.match({ name: "John", age: "25" })  ==> false
c.match({ name: "John" })  ==> false
