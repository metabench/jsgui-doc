Class: Int_Constraint

core/constraint. Int_Constraint

Int field constraint: checks value to be an integer number.

new Int_Constraint()

The constructor is private. Use from_obj() or from_str() to create this constraint.

Methods

match(value)

Returns true if the value matches the constraint (i.e. value is an integer number).

Parameters:
Name Type Description
value *

value to check
Example
var c = Constraint.from_obj("int");
c.match(1)  ==> true
c.match(1.5)  ==> false
c.match("1")  ==> false

to_info_obj()

Returns the constraint information object (i.e. "int" string).

Example
var c = Constraint.from_obj("int");
c.to_info_obj()  ==> "int"
