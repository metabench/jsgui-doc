<static> from_str(str)

Creates a constraint using the definition string. The following definitions are supported:

  • "int" - creates Int_Constraint
  • "number" - creates Number_Constraint
  • "text" - creates Text_Constraint without max length restriction
  • "text(<length>)" - creates Text_Constraint, where "<length>" is the max text length allowed
  • "guid" - creates Guid_Constraint
Parameters:
Name Type Description
str string

constraint definition
Example
c = Constraint.from_str("int");
c = Constraint.from_str("text(10)");

<static> obj_matches_constraint(obj, constraint)

Returns true if the obj value matches the constraint definition.

Parameters:
Name Type Description
obj object

object to check
constraint string

constraint definition
Example
obj_matches_constraint(1, "int") ==> true
obj_matches_constraint("1", "int") ==> false
obj_matches_constraint("1", "text(3)") ==> true
