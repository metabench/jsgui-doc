<static> Class

Base class like John Resig's "Simple JavaScript Inheritance" tehnique.
In addition, when the inherited class contains a property that name starts from '#', then the property adds to the class without '#', and the property value is the base class property value, but the base class property name used is the rest of the initial property name (after the '#').
See:
  • John Resig's "Simple JavaScript Inheritance"

<static> each(collection, fn, context)

Iterates over a collection of elements. Each element is passed to an iterator function. The iterator function is bound to the context object, if passed. The collection parameter can be: - a Collection object (the Collection.each() method is used in this case) - an array - an object The iteration function parameters are: index, element, stopFunction (for array) key, value, stopFunction (for object) The stopFunction breaks the iteration if called.
Parameters:
Name Type Argument Description
collection Collection | Array | Object collection of elements
fn module:core/jsgui-lang-essentials.arrayIteratorCallback | module:core/jsgui-lang-essentials.objectIteratorCallback iterator function
context Object <optional>
 context object
